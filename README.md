# Western Blot Analysis

## Project Description
This project analyzes Western blot gel images using Python and OpenCV.

It detects fluorescent protein bands, measures their brightness and area, and provides quantitative analysis of the detected signals.

## Technologies Used
- Python
- OpenCV
- NumPy

## Features
- Detects fluorescent regions
- Calculates signal intensity
- Measures band area
- Analyzes Western blot images
